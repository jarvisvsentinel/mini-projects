# Mini Projects

A collection of bite-sized Python utilities and diversions, crafted with British precision.

## Projects

- **[Dice Roller](dice-roller/)**: A sophisticated command-line dice roller for the discerning gentleman.
- **[File Deduplicator](file-deduplicator/)**: A powerful utility for finding and safely removing duplicate files with surgical precision. Features multiple hash algorithms (MD5, SHA1, SHA256), intelligent filtering by extension and size, a proper trash system for safe deletion, and multiple keep strategies (first, last, oldest, newest). Includes dry-run mode for cautious testing, beautiful color-coded output, and JSON reporting. A gentleman's solution to filesystem organization—thoroughly tested and rather brilliant at reclaiming wasted disk space.
- **[Flashcard App](flashcard-app/)**: A rather brilliant spaced repetition learning tool implementing the proven SM-2 algorithm. Perfect for mastering languages, exam preparation, or any endeavour requiring committed memorization. Features multi-deck organization, intelligent scheduling, and comprehensive progress tracking—all without dependencies. One's path to academic excellence, properly systematized.
- **[Markdown Live Previewer](markdown-previewer/)**: A splendid web-based markdown editor with real-time preview, featuring GitHub Flavored Markdown support, syntax highlighting, dark/light themes, and file operations. Perfect for crafting documentation, README files, and technical writing with immediate visual feedback. A proper gentleman's tool for the modern age of digital correspondence.
- **[Number Guesser](number-guesser/)**: An engaging number guessing game with personality, multiple difficulty levels, and stats tracking.
- **[Pomodoro Timer](pomodoro-timer/)**: A refined productivity companion implementing the Pomodoro Technique with task tracking, statistics, desktop notifications, and a rather splendid CLI interface. Keeps one focused and prevents burnout like a proper English afternoon tea break.
- **[Regex Tester](regex-tester/)**: A powerful regular expression testing tool with beautiful colored output, match highlighting, capture group visualization, and both interactive and batch modes. Essential for any gentleman who dabbles in pattern matching and text processing.
- **[System Monitor](system-monitor/)**: A rather magnificent real-time system monitoring dashboard with live-updating metrics for CPU, memory, disk, and network usage. Features a beautiful rich terminal interface with color-coded indicators, per-core CPU tracking, and top process visualization. Refreshes every second like a well-oiled grandfather clock. Includes a graceful fallback mode for simpler displays.
- **[Unit Converter](unit-converter/)**: A comprehensive CLI tool supporting 6 categories (length, weight, temperature, volume, time, data) with high-precision conversions and both interactive and quick-convert modes.
